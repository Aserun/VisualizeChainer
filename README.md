
# TkinterChainer
chainerで学習させたニューラルネットワークを実際に使ってみる

## MNIST
マウス入力できるcanvasはTkinterで作成されています。
canvasのピクセルデータを学習済みのニューラルネットワークに入力して数字を認識します。

    1. Tkinterのcanvasに数字を書く
    2. judgeボタンを押すと数字を認識して結果を右側に表示される
